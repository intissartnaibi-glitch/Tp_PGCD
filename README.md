# Tp_PGCD  
♡ 1.définition :
le PGCD de deux nombres est le plus grand nombre qui divise les deux sans reste .
___
♡ 2. algorithme d’euclide : 
algorithme algorithme d’euclide 
variable a,b ,PGCD:entier
début 
Ecrire("Entrez deux nombres entiers")
lire (a ,b)
Tant que b≠0
r recoit a mod b
a recoit b 
b recoit r 
Écrire ("le PGCD est : ",a)
fin 
____
